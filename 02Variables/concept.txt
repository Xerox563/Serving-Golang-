02Variables Module Concepts
========================

Key Concepts:
1. Variable Declaration
   - Using 'var' keyword
   - Short declaration operator (:=)
   - Global vs local scope
   - Constants (const keyword)

2. Data Types
   - string
   - bool
   - int/uint (including sized variants like uint8)
   - float32/float64
   - Default values for types

3. Important Features Demonstrated:
   - Variable type inference
   - Printf formatting with %T for type printing
   - Public vs private variables (LoginToken as public)
   - Constants
   - Global variable declaration
   - Walrus operator limitations (can't use outside functions)

4. Variable Declaration Methods:
   - Explicit type declaration: var x int = 34
   - Type inference: var web = "alphaxerox.com"
   - Short declaration: numberofuser := 3000.1
   - Constant declaration: const LoginToken string = "..."

5. Numeric Types:
   - Integer types (int, uint8)
   - Floating point precision (float32 vs float64)

This module covers fundamental variable concepts and type system in Go.
